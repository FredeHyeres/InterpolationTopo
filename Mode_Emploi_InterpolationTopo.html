📏 Interpolation Topo

Macro VBA pour MicroStation V8i SS3 — interpolation d'altitude entre deux points topo

Présentation Installation Utilisation Key-in Touche de fonction Boîte à outils

1. Présentation

Interpolation Topo calcule et place l'altitude d'un point situé sur la droite reliant deux textes d'altitude existants (points topo). La macro crée :

  • un texte altitude reprenant automatiquement la mise en forme (police, taille, niveau, couleur) du texte du point 1 pris comme modèle ;
  • un cercle centré sur le point projeté, avec diamètre, couleur et niveau au choix.

Pendant le placement, un aperçu dynamique affiche :

  • la ligne support P1–P2 (verte, étendue de 20 % de part et d'autre) ;
  • des croix sur P1 et P2 ;
  • le cercle et le texte altitude provisoires qui suivent le curseur ;
  • dans la barre d'état : l'altitude interpolée Z, l'abscisse t (0 = P1, 1 = P2), la pente en % et le gisement en gon (0 = nord, sens horaire).
✅ Le séparateur décimal (point ou virgule) et le nombre de décimales du texte créé sont repris automatiquement des textes source.

2. Installation

2.1 Créer le projet VBA

  1. Dans MicroStation : Utilitaires > Macros > Gestionnaire de projets VBA
  2. Bouton Nouveau projet : nommez-le InterpolationTopo (MicroStation crée le fichier .mvba)
  3. Ouvrez l'éditeur VBA : Alt+F11

2.2 Importer les 4 fichiers sources

  1. Dans l'éditeur VBA : Fichier > Importer un fichier (Ctrl+M)
  2. Importez, un par un, depuis le dossier src\ :
    FichierType attendu après import
    InterpolationTopo.basModules
    CSelectP1.clsModules de classe
    CSelectP2.clsModules de classe
    CPlacerPoint.clsModules de classe
  3. Débogage > Compiler InterpolationTopo : aucune erreur ne doit apparaître
  4. Enregistrez (Ctrl+S)
⚠️ Les fichiers .cls doivent arriver dans « Modules de classe ». S'ils arrivent dans « Modules » avec des lignes VERSION 1.0 CLASS ou Attribute VB_Name visibles, les fichiers sources ont des fins de ligne Unix (LF) : convertissez-les en fins de ligne Windows (CRLF) puis réimportez.

2.3 Chargement automatique (recommandé)

Pour que la macro soit disponible sans manipulation :

  1. Placez le fichier .mvba dans un dossier scruté par MicroStation, par exemple ...\Workspace\Standards\vba\ (ou tout dossier listé dans la variable MS_VBASEARCHDIRECTORIES)
  2. Facultatif — chargement au démarrage : Espace de travail > Configuration, variable MS_VBAAUTOLOADPROJECTS, valeur InterpolationTopo

  1. Paramètres (3 petites boîtes de dialogue) :
    • Diamètre du cercle en unités maître (ex. 0.50)
    • Couleur du cercle : index MicroStation 0–255 (0=blanc, 1=bleu, 2=vert, 3=rouge, 4=jaune, 6=orange)
    • Niveau du cercle : la liste des niveaux existants du fichier s'affiche (« numéro : nom ») — tapez le numéro ou le nom voulu ; laisser vide pour reprendre le niveau du texte P1. Une saisie invalide fait réafficher la liste.
      Utile pour placer les cercles sur un niveau distinct des textes, afin d'imprimer les altitudes sans les symboles de points.
  2. Point 1 : cliquez à moins de 1 unité maître du texte altitude. Ce texte sert de modèle de mise en forme pour le texte créé.
  3. Point 2 : idem. La barre d'état affiche alors la pente et le gisement du segment.
  4. Placement : déplacez le curseur — sa position est projetée sur la droite P1–P2 et l'altitude interpolée s'affiche en dynamique.
    • Data (clic gauche) : crée le point (texte + cercle). Répétable : on peut créer plusieurs points sur la même droite.
    • Reset (clic droit) : nouvelle sélection — on revient au choix du point 1 pour travailler sur une autre paire de points (les paramètres du cercle sont conservés).
    • Reset une seconde fois (pendant le choix du point 1) : quitte la commande.
⚠️ Si t < 0 ou t > 1, le point est hors du segment P1–P2 : l'altitude est extrapolée, un avertissement s'affiche dans la barre d'état.
✅ Seuls les textes numériques sont détectés : les matricules contenant des lettres sont ignorés automatiquement lors de la recherche du texte le plus proche.

4. Lancement par key-in

  1. Ouvrez la fenêtre de saisie : Utilitaires > Saisie au clavier (Key-in)
  2. Tapez : 
    vba run [InterpolationTopo]InterpolerPoint
  3. Validez par Entrée
✅ La fenêtre Key-in garde l'historique : ↑ puis Entrée pour relancer la dernière commande. Si le projet n'est pas chargé, vba run le charge automatiquement à condition qu'il soit dans un dossier de MS_VBASEARCHDIRECTORIES (voir § 2.3).

5. Lancement par touche de fonction

  1. Menu Utilitaires > Touches de fonction
  2. Sélectionnez une touche libre, par exemple F6 (les combinaisons Ctrl/Alt/Maj+F sont aussi possibles)
  3. Dans le champ Action, saisissez : 
    vba run [InterpolationTopo]InterpolerPoint
  4. Cliquez OK — le menu de touches de fonction est enregistré (fichier .mnu du workspace)

Un appui sur F6 lance désormais directement la commande.

6. Bouton dans une boîte à outils personnalisée

La méthode la plus confortable pour un usage quotidien. Les personnalisations sont enregistrées dans une bibliothèque DGN (.dgnlib) et peuvent donc être partagées entre plusieurs postes.

  1. Menu Espace de travail > Personnaliser (Workspace > Customize)
  2. Onglet Outils. Si demandé, acceptez la création de la bibliothèque Personal.dgnlib
  3. Dans l'arborescence, clic droit > Nouvelle boîte à outils : nommez-la par exemple Topo
  4. Clic droit sur la boîte Topo > Nouvel outil : nommez-le Interpolation altitude
  5. Dans les propriétés de l'outil :
    PropriétéValeur
    Key-invba run [InterpolationTopo]InterpolerPoint
    Infobulle (Balloon)Interpolation d'altitude entre 2 points topo
    Icôneà dessiner dans l'éditeur d'icônes intégré, ou réutiliser une icône existante
  6. Fermez la boîte de dialogue Personnaliser
  7. Affichez la boîte à outils : Outils > Boîtes à outils (ou clic droit sur une barre d'outils existante), cochez Topo et ancrez-la où vous voulez
✅ Partage : placez la .dgnlib dans le dossier standards du workspace (variable MS_DGNLIBLIST) pour que la boîte à outils soit disponible sur tous les postes qui utilisent ce workspace.

7. Dépannage rapide

SymptômeCause / remède
« Ouvrez d'abord un fichier DGN » Aucun dessin actif : ouvrez un DGN avant de lancer la commande.
« Aucun texte altitude à moins de 1 u.m. » Cliquez plus près de l'origine du texte, ou le texte n'est pas numérique (lettres, symboles).
Pas d'aperçu dynamique Vérifiez que la version chargée de CPlacerPoint contient bien CommandState.StartDynamics dans Start (supprimer le module puis réimporter le fichier).
Erreurs de compilation « membre introuvable » Vérifiez Outils > Références : la « Bentley MicroStation DGN 8.9 Object Library » doit être cochée et pointer vers l'installation V8i active.
Les .cls s'importent comme modules standard Fins de ligne Unix (LF) dans les fichiers : convertir en CRLF puis réimporter.
« P1 et P2 confondus » Les deux clics ont attrapé le même texte : cliquez plus près du second point.
