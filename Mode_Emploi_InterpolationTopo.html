1. Présentation

Interpolation Topo V2 propose deux commandes pour le calcul et le placement d'altitudes topographiques :

CommandeKey-inDescription
Interpolation vba run [InterpolationTopoV2]InterpolerPoint Interpole l'altitude sur la droite P1–P2, avec pente transversale, DZ et indicateur de pente
Interpol. Ponctuelle vba run [InterpolationTopoV2]InterpolPonctuelle Chemin principal depuis P1 (pente/DZ cumulatifs) + rayonnement latéral depuis chaque point

Sources d'altitude (communes aux deux commandes)

  • des textes simples contenant une valeur numérique ;
  • des tags et cellules contenant des tags d'altitude ;
  • des éléments provenant de fichiers de référence attachés (les coordonnées sont converties automatiquement dans le repère du modèle actif).

Interpolation (P1–P2)

Calcule et place l'altitude d'un point sur la droite reliant deux points existants. Elle crée :

  • un texte altitude reprenant automatiquement la mise en forme (police, taille, niveau, couleur) du texte du point 1 pris comme modèle ;
  • un cercle centré sur le point projeté, avec diamètre, couleur et niveau au choix ;
  • un indicateur de pente optionnel : texte pente signée (+/−) et flèche dirigée de P1 vers P2.

Pendant le placement, un aperçu dynamique affiche :

  • la ligne support P1–P2 (étendue de 20 % de part et d'autre) ;
  • des croix sur P1 et P2 ;
  • le Z sur la droite (en vert, au point projeté) et le Z final (avec pente/DZ appliqués, au point déporté) ;
  • dans la barre d'état : l'altitude interpolée Z, l'abscisse t (0 = P1, 1 = P2), la pente en % et le gisement en gon (0 = nord, sens horaire).

Interpolation Ponctuelle (chemin + rayonnement)

Construit un chemin principal depuis P1, puis permet des rayonnements latéraux :

  • P2 optionnel : s'il est sélectionné, la pente P1–P2 est calculée automatiquement ; sinon, saisie manuelle dans le formulaire ;
  • Chemin principal : ligne verte dynamique, Z = Z précédent + pente × distance + DZ ;
  • Rayonnement : après chaque point chemin, ligne jaune dynamique avec pente/DZ propres au rayonnement ;
  • Retour au chemin : clic droit en mode rayonnement reprend le chemin depuis le dernier point créé.

Les éléments créés sont positionnés en 3D à l'altitude calculée (coordonnée Z). L'aperçu dynamique affiche 3 décimales pour le contrôle visuel ; le nombre de décimales des points créés est configurable (0–6).

✅ Le séparateur décimal (point ou virgule) et le nombre de décimales du texte créé sont repris automatiquement des textes source (modifiables dans le formulaire).

2. Installation

2.0 Installation automatique (recommandée)

Télécharger le dépôt GitHub : téléchargement direct du ZIP (ou Code > Download ZIP sur la page du dépôt), décompresser, puis double-cliquer sur install.cmd : le projet Interpolation.mvba est copié dans le workspace avec chargement automatique au démarrage, et la ToolBox MesMacros.dgnlib est installée. Le Default.mvba personnel n'est jamais touché. Redémarrer MicroStation et c'est prêt. Les étapes 2.1 et 2.2 ci-dessous ne servent que pour une installation manuelle.

2.1 Ouvrir le projet VBA (installation manuelle)

  1. Dans MicroStation : Utilitaires > Macros > Gestionnaire de projets VBA
  2. Utilisez le projet Default (chargé automatiquement au démarrage) — ou créez un projet dédié si vous préférez
  3. Ouvrez l'éditeur VBA : Alt+F11

2.2 Importer les fichiers sources

  1. Dans l'éditeur VBA : Fichier > Importer un fichier (Ctrl+M)
  2. Importez, un par un, depuis le dossier src\ :

⚠️ Les fichiers .frx (frmInterpolation.frx et frmInterpolPonct.frx) doivent être dans le même dossier que leur .frm respectif lors de l'import. Sans eux, l'import échoue avec « Impossible de charger... ».

2.3 Chargement automatique

Avec le Default MVBA : rien à faire, il est chargé d'office par MicroStation à chaque démarrage.

Avec un .mvba dédié seulement : Utilities > Macros > Project Manager, clic droit sur le projet > Autoload.

3. Formulaire Interpolation (Tool Settings)

Au lancement de la commande, le formulaire Interpolation Topo s'ouvre et reste visible pendant toute la durée de la commande. Les paramètres sont modifiables à tout moment, même entre deux placements.

Cercle (si l'altitude est un texte)

  • Diamètre en unités maître (ex. 0.50)
  • Couleur : index MicroStation 0–255 (0=blanc, 1=bleu, 2=vert, 3=rouge, 4=jaune, 6=orange)
  • Niveau : liste des niveaux existants du fichier DGN. Laisser vide pour reprendre le niveau du texte P1

Texte altitude (si l'altitude est un texte)

  • Mêmes attributs que le texte P1 (case cochée par défaut) : couleur, niveau, police, taille, style et justification sont hérités du texte P1
  • Case décochée : la couleur et le niveau du texte créé deviennent choisissables (les autres propriétés restent héritées)

Pente décalage (pente transversale)

  • Appliquer pente transversale : cocher la case pour activer
  • Pente (%) : valeur de la pente. Positive = montée, négative = descente. La pente s'applique identiquement des deux côtés de la ligne P1–P2
  • Bouton +/- : inverse le signe de la pente en un clic
  • Décalage fixe DZ : cocher la case pour activer. La valeur (en unités maître) est ajoutée à l'altitude interpolée

Indicateur pente

  • Personnaliser : cocher la case pour définir manuellement les propriétés du texte de pente et de la flèche. Sans personnalisation, les attributs sont repris du texte modèle
  • H / L : hauteur et largeur du texte de pente
  • Niv : niveau (liste des niveaux DGN)
  • C.txt : couleur du texte de pente (index 0–255)
  • C.fl : couleur de la flèche (index 0–255)
  • Long : longueur de la flèche (en unités maître)

État (lecture seule)

  • P1 : altitude du premier point sélectionné (affichée dès la sélection)
  • P2 : altitude du second point sélectionné (affichée dès la sélection)
  • Pente (%) et gisement (gon) du segment

Actions

  • Interpolation : bascule vers le mode placement de points interpolés (actif après snap P2)
  • Pente + flèche : bascule vers le placement de l'indicateur de pente (actif après snap P2)

  1. Lancement : le formulaire Interpolation Topo s'ouvre. Tous les paramètres sont modifiables à tout moment via le formulaire.
  2. Sélection P1 : cliquez à proximité d'un texte, tag ou cellule d'altitude. Ce texte sert de modèle de mise en forme pour le texte créé. L'altitude s'affiche immédiatement dans le formulaire. Les niveaux gelés sont automatiquement ignorés. Les références attachées sont également recherchées.
  3. Snap terrain P1 : cliquez sur le point terrain réel pour positionner P1 (le snap MicroStation est utilisé). Reset = garder l'ancrage du texte comme position terrain.
  4. Sélection P2 : idem que P1. Le formulaire affiche alors la pente et le gisement du segment. Les boutons Interpolation et Pente + flèche deviennent actifs.
  5. Snap terrain P2 : idem que pour P1.
  6. Placement : déplacez le curseur — sa position est projetée sur la droite P1–P2 et l'altitude interpolée s'affiche en dynamique.
    • Data (clic gauche) : crée le point (texte + cercle) à l'altitude interpolée en 3D. Répétable : on peut créer plusieurs points sur la même droite.
    • Reset (clic droit) : nouvelle sélection — on revient au choix du point 1 pour travailler sur une autre paire de points (les paramètres sont conservés).
    • Reset une seconde fois (pendant le choix du point 1) : quitte la commande.
⚠️ Si t < 0 ou t > 1, le point est hors du segment P1–P2 : l'altitude est extrapolée, un avertissement s'affiche dans la barre d'état.
✅ Seuls les éléments contenant une valeur numérique sont détectés : les matricules contenant des lettres sont ignorés automatiquement.
✅ La pente transversale s'applique de manière uniforme des deux côtés de la ligne P1–P2 : une valeur positive fait monter, une valeur négative fait descendre, quel que soit le côté. Utilisez le bouton +/- pour inverser rapidement.
✅ Le décalage fixe DZ s'ajoute à l'altitude interpolée et à la pente transversale. Utile pour un décalage constant (ex. épaisseur de chape, arase).

5. Indicateur de pente

Après sélection et snap des deux points, le bouton Pente + flèche dans le cadre Actions bascule vers le placement de l'indicateur de pente.

  • Un texte affiche la pente signée : +2,50% (montée) ou -1,30% (descente)
  • Une flèche dirigée de P1 vers P2, orientée parallèlement au segment
  • Le texte est justifié en bas à gauche, au-dessus de la flèche
  • Data = placer l'indicateur (répétable)
  • Reset = nouvelle sélection de points

Personnalisation

Cocher Personnaliser dans le cadre Indicateur pente du formulaire pour définir manuellement la hauteur/largeur du texte, les couleurs (texte et flèche), la longueur de la flèche et le niveau. Sans personnalisation, les propriétés sont héritées du texte modèle (P1).

✅ Le bouton Interpolation permet de revenir au mode placement de points interpolés sans refaire la sélection P1/P2.

  1. Lancement : le formulaire Interpol. Ponctuelle s'ouvre. Réglez les paramètres chemin et rayonnement avant ou pendant l'utilisation.
  2. Sélection P1 : cliquez près d'un texte/tag/cellule d'altitude. Ce texte sert de modèle de mise en forme. Reset = quitter.
  3. Snap terrain P1 : cliquez sur le point terrain réel. Reset = garder l'ancrage du texte.
  4. P2 optionnel :
    • Data = sélectionner un texte P2 (la pente P1–P2 est calculée automatiquement et reportée comme pente du chemin principal)
    • Reset = sauter P2, utiliser la pente saisie manuellement dans le formulaire, démarrer le chemin directement
  5. Mode Chemin (ligne verte) :
    • Une ligne dynamique relie le dernier point au curseur
    • L'altitude affichée = Z précédent + pente × distance + DZ
    • Data = créer le point (cercle + texte)
    • Après création, passage automatique en mode Rayonnement
    • Reset = nouvelle sélection P1
  6. Mode Rayonnement (ligne jaune) :
    • Une ligne dynamique relie le point chemin (centre) au curseur
    • L'altitude = Z du centre + pente rayon × distance + DZ rayon
    • Data = créer un point rayon (répétable : autant de rayons que souhaité)
    • Reset = terminer le rayonnement, repartir en chemin depuis le dernier point chemin créé
✅ Pour les cellules/tags intégrés (source P1), le texte est placé automatiquement dans la cellule — les étapes de placement libre du texte sont sautées.
✅ L'aperçu dynamique affiche 3 décimales pour le contrôle visuel. Les points créés utilisent le nombre de décimales configuré dans le formulaire.

7. Formulaire Interpol. Ponctuelle

Au lancement de la commande Ponctuelle, le formulaire Interpol. Ponctuelle s'ouvre. Les paramètres sont modifiables à tout moment.

Chemin principal

  • Pente (%) : cocher la case pour activer. Valeur cumulative appliquée proportionnellement à la distance parcourue. Bouton +/- pour inverser
  • DZ : cocher la case pour activer. Décalage fixe ajouté à chaque point
  • Si P2 est sélectionné, la pente P1–P2 est calculée automatiquement et reportée ici

Rayonnement

  • Pente (%) : pente propre au rayonnement (indépendante du chemin). Bouton +/- pour inverser
  • DZ : décalage fixe propre au rayonnement

Décimales

  • Nombre de décimales (0–6) pour les points créés, commun au chemin et au rayonnement

Cercle / Texte altitude / État

Identiques à la commande Interpolation (voir § 3).

8. Lancement par key-in

  1. Ouvrez la fenêtre de saisie : Utilitaires > Saisie au clavier (Key-in)
  2. Tapez l'un des key-in suivants :
    CommandeKey-in
    Interpolationvba run [InterpolationTopoV2]InterpolerPoint
    Interpol. Ponctuellevba run [InterpolationTopoV2]InterpolPonctuelle
  3. Validez par Entrée
✅ La fenêtre Key-in garde l'historique : ↑ puis Entrée pour relancer la dernière commande.

9. Lancement par touche de fonction

  1. Menu Utilitaires > Touches de fonction
  2. Sélectionnez deux touches libres, par exemple F6 et F7 (les combinaisons Ctrl/Alt/Maj+F sont aussi possibles)
  3. Dans le champ Action, saisissez pour chaque touche :
    ToucheKey-in
    F6vba run [InterpolationTopoV2]InterpolerPoint
    F7vba run [InterpolationTopoV2]InterpolPonctuelle
  4. Cliquez OK — le menu de touches de fonction est enregistré (fichier .mnu du workspace)

Un appui sur F6 lance l'Interpolation, F7 lance l'Interpol. Ponctuelle.

10. Boutons dans une boîte à outils personnalisée

La méthode la plus confortable pour un usage quotidien. Procédure testée et validée sur V8i SS3 (08.11.09.578).

⚠️ Piège n°1 : dans Workspace > Customize, si les menus sont grisés et qu'aucune création n'est possible, c'est qu'il manque un fichier .dgnlib valide comme cible des personnalisations.
Piège n°2 : un .dgnlib créé via File > New (ou en renommant l'extension d'un .dgn) n'est pas reconnu — sa structure interne est incorrecte. Il faut copier un .dgnlib existant.

Étape 1 — Créer un vrai fichier .dgnlib

  1. Aller dans le dossier scanné automatiquement par MicroStation :
    C:\Users\[VotreNom]\Documents\MicroStV8i\WorkSpace\System\GUI
  2. Copier le fichier VectorIcon.dgnlib (structure interne valide)
  3. Renommer la copie en MesMacros.dgnlib
  4. Redémarrer MicroStation

Étape 2 — Vérifier que le fichier est reconnu

  1. Workspace > Customize : MesMacros.dgnlib doit apparaître dans la liste des librairies
  2. Sinon, vérifier qu'il est bien dans System\GUI

Étape 3 — Ouvrir le .dgnlib pour le rendre éditable

  1. File > Open, changer le filtre de type pour afficher les .dgnlib
  2. Ouvrir MesMacros.dgnlib comme fichier actif
  3. Retourner dans Workspace > Customize : les menus sont maintenant déverrouillés

Étape 4 — Créer la ToolBox

  1. Sélectionner MesMacros.dgnlib comme fichier cible (menu déroulant en haut)
  2. Onglet Tool Boxes > clic droit dans la liste > New Tool Box, nommer (ex. MesMacros)

Étape 5 — Créer les boutons

Répéter l'opération pour chaque commande :

  1. Sélectionner la ToolBox, panneau de droite > clic droit > New Tool
  2. Remplir pour le premier bouton :
    PropriétéValeur
    Key-invba run [InterpolationTopoV2]InterpolerPoint
    Balloon textInterpoler un point (V2)
    Iconlaisser vide ou choisir une icône
  3. Clic droit > New Tool pour le second bouton :
    PropriétéValeur
    Key-invba run [InterpolationTopoV2]InterpolPonctuelle
    Balloon textInterpol. Ponctuelle (chemin + rayon)
    Iconlaisser vide ou choisir une icône

Étape 6 — Afficher la ToolBox

  1. Clic droit sur la ToolBox dans la liste > Open : elle apparaît flottante
  2. La docker contre une barre d'outils existante par glisser-déposer

Étape 7 — Autoload de la macro (indispensable)

  1. Utilities > Macros > Project Manager
  2. Charger le projet VBA s'il ne l'est pas déjà
  3. Clic droit sur le projet > Autoload : la macro sera chargée à chaque ouverture de MicroStation et le bouton fonctionnera sans manipulation

11. Dépannage rapide

SymptômeCause / remède
« Ouvrez d'abord un fichier DGN » Aucun dessin actif : ouvrez un DGN avant de lancer la commande.
« Aucun texte altitude à proximité » Cliquez plus près de l'origine du texte/tag, ou l'élément n'est pas numérique. Vérifiez aussi que le niveau n'est pas gelé.
Pas d'aperçu dynamique Vérifiez que la version chargée de CPlacerPoint contient bien CommandState.StartDynamics dans Start (supprimer le module puis réimporter le fichier).
Erreurs de compilation « membre introuvable » Vérifiez Outils > Références : la « Bentley MicroStation DGN 8.9 Object Library » doit être cochée et pointer vers l'installation V8i active.
Les .cls s'importent comme modules standard Fins de ligne Unix (LF) dans les fichiers : convertir en CRLF puis réimporter.
« P1 et P2 confondus » Les deux clics ont attrapé le même texte : cliquez plus près du second point.
Import du .frm échoue Le fichier .frx correspondant doit être dans le même dossier que le .frm (frmInterpolation.frx ou frmInterpolPonct.frx). En dernier recours : Insertion > UserForm, nommer le formulaire, puis coller le code à partir de Option Explicit — les contrôles sont créés au runtime.
Menus grisés dans Workspace > Customize Aucun .dgnlib cible valide. Copier VectorIcon.dgnlib (dossier WorkSpace\System\GUI), renommer la copie (ex. MesMacros.dgnlib), redémarrer, puis l'ouvrir comme fichier actif pour déverrouiller Customize (voir § 8).
Le bouton de la ToolBox ne fait rien au démarrage Le projet VBA n'est pas chargé : Utilities > Macros > Project Manager, clic droit sur le projet > Autoload.
Taille texte pente H:1 / L:1 avec des tags Le TagElement VBA V8i n'expose pas .TextStyle. Cocher Personnaliser dans le cadre Indicateur pente du formulaire et régler manuellement la hauteur et la largeur du texte.
